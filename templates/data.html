How the workflow works

  1. Capture in the app — log incidents, near misses/hazards and corrective actions from the buttons in the top bar; they write straight to the database and appear instantly.
  2. Or bulk-load from Excel — drop spreadsheets in {{ status.data_dir }} (same headers) and an admin clicks Import from Excel.
  3. Refresh re-reads the database; Export downloads any dataset to Excel.
  4. Need fresh sample files? Run python generate_dummy_data.py.
